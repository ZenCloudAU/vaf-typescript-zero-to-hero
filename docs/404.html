Curriculum

What this course teaches

This course helps practitioners move from vibe coding to controlled delivery. It teaches enough TypeScript to read generated code, run it safely, debug it, and understand how data contracts support architecture governance.

Practice

Programming literacy

Values, variables, functions, objects, arrays, and output.

Governance

Type safety

Types and interfaces as control points for AI-generated code.

Application

Decision models

Small scoring tools aligned to Velocity Architecture principles.

Run the course locally

The course runs from the command line. Every lesson is a standalone TypeScript file that can be inspected, executed, changed, and committed.

git clone https://github.com/ZenCloudAU/vaf-typescript-zero-to-hero.git
cd vaf-typescript-zero-to-hero
npm install
npm run check
npm run lesson:all
