TypeScript Zero to Hero

A hands-on course that teaches the programming language behind AI-native architecture tools, decision engines, artefact generators, and Velocity Academy learning products.

View lessons GitHub repository
Overview

Built for architects who are learning to control generated code.

This course is not abstract programming theory. Every lesson runs locally, prints output, and connects programming concepts back to architecture: contracts, artefacts, decision rules, governance, quality gates, and AI delivery guardrails.

Foundation

Programming basics

Values, variables, functions, objects, arrays, and terminal output.

Control

TypeScript contracts

Types and interfaces that define what valid data must look like.

Application

VAF decision models

Small decision-support examples using value, risk, confidence, and readiness.

Run locally

Run the entire course from the terminal.

Each lesson is a standalone TypeScript file. The full course runs from Lesson 00 through Lesson 12.

git clone https://github.com/ZenCloudAU/vaf-typescript-zero-to-hero.git
cd vaf-typescript-zero-to-hero
npm install
npm run check
npm run lesson:all

Lesson 00

Programming Primer

Introduces programming as data plus instructions. Covers strings, numbers, booleans, variables, functions, objects, and console output.

  • Recognise values and variables.
  • See how a function creates an output.
