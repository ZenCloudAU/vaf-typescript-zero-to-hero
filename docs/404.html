Course purpose

This course teaches enough TypeScript to control AI-generated software. It is not a traditional computer science course. It is a practical bridge between architecture thinking, coding literacy, and product delivery.

For architects

Understand types as contracts, interfaces as artefact structures, and the compiler as a quality gate.

For vibe coders

Move from guessing and prompting into reading, checking, editing, and shipping safely.

For StudioSix

Create a reusable learning product that supports Academy, advisory, and AI tooling work.

Learning path

Thirteen short lessons move from basic programming concepts into a small VAF decision-support model.

Learning loop

Read one lesson. Run it. Change one thing. Run it again. Commit the milestone.

Architecture lens

Type

A rule that says what kind of value is allowed.

Interface

A contract that defines the required shape of an artefact.

Function

A reusable decision rule that converts inputs into outputs.

Compiler

A quality gate that catches structural errors before runtime.

Outcome

By the end, learners can inspect generated TypeScript, understand the shape of the data, identify unsafe changes, and build small decision-support tools aligned to Velocity Architecture principles.
