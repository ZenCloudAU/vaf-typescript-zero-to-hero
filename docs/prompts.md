# AI Coding Prompts

## Explain

```text
Explain this TypeScript file to a beginner. Focus on data inputs, logic, and outputs.
```

## Debug

```text
Find the smallest possible fix for this TypeScript error. Explain the cause before changing code.
```

## Refactor

```text
Refactor this code for clarity. Do not change the external behaviour. Preserve all function names and return types.
```

## Extend

```text
Add one new feature to this file using the same style. Keep it beginner-friendly.
```

## Test understanding

```text
Ask me five questions that check whether I understand this TypeScript lesson.
```
